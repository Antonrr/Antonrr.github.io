Armored Warfare: Assault

June 2016–January 2019

Role: Lead game client developer

Armored Warfare: Assault is a 3D online synchronous PVP tank battle game with AAA graphics and real-world-like physics running on Unreal Engine 4. On this project I took part in developing various aspects of vehicle simulation model, in-game skills and abilities, tutorial system, bots, and more. I also managed iOS platform-specific features and contributed to the engine. The game was released on Android and iOS platforms in mid 2018.
